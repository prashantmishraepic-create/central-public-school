# Central Public School - Complete Educational Website System

## Overview
A comprehensive, professional-grade educational institution website with integrated Student Portal and Admin Panel management system for Central Public School, Azamgarh.

## Institution Details
**Name:** Central Public School  
**Location:** Near Kanshiram Awas Colony, Jaffarpur, Azamgarh, Uttar Pradesh 276128  
**Type:** CBSE Affiliated School  
**Classes Offered:** Pre-Primary to Class XII  
**Streams:** Science, Commerce, Arts

### Changing School Information
1. Edit school name in `index.html`
2. Update contact details in contact section
3. Modify address in footer
4. Update logo/emblem references

### Modifying Colors
1. Edit CSS variables in `:root` section
2. Update gradient colors in respective CSS files
3. Adjust brand colors in all three stylesheets

### Adding New Sections
1. Create HTML section in appropriate file
2. Add corresponding CSS styles
3. Update navigation menu
4. Add JavaScript functionality if needed
